# Pasword_Generator  
(EN)  
The password generator generates completely different passwords. You can also choose which characters the password will contain: 
1. Only numbers
2. Only letters
3. Numbers + letters
4. Numbers + letters + symbols.  
You can also set the password length yourself.
# 
(RU)  
Генератор паролей, генерирует абсолютно разные пароли, так же есть выбор из каких символов будет состоять пароль: 
1. только цифры,
2. только буквы,
3. цифры + буквы,
4. цифры + буквы + символы.  
Так же длинну пароля вы задаете сами.
